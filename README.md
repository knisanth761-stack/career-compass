# 🚀 Career Compass

## 📖 Problem Statement

Many students struggle with:

- Not knowing where to start in IT
- Learning random technologies without direction
- Losing motivation after a few days
- Having no clear idea which companies they are ready for

This leads to wasted time, confusion, and lack of consistency.

---

## 💡 Solution

Career Compass is a platform that helps students:

✅ Choose a clear career roadmap  
✅ Track daily progress in a structured way  
✅ Maintain consistency with streak tracking  
✅ Receive realistic motivation based on actual progress  
✅ Discover companies they can realistically target  

---

## ✨ Key Features (Planned)

- Career roadmap selection (DSA, Web Dev, Data, etc.)
- Daily progress tracking system
- Consistency and momentum scoring engine
- Company Reach Meter (realistic eligibility tracking)
- Smart recommendations for next learning steps
- Resume-ready progress analytics

---

## 🧠 System Design (High Level)

User → Logs Progress → Engine Calculates Scores → Platform Recommends Next Steps → Shows Eligible Companies
